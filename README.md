## file explain

.babelrc ------ ES6转ES5，兼容IE

.gitignore ------ 忽略 git 版本提交的文件

package.json ------ 管理你本地安装的 npm 包

webpack.config.js ------ 管理 webpack 的参数设置

client ------ 项目文件

dist ------ webpack打包生成目录

## command

webpack ------ 打包创建dist目录下的文件

npm run start ------ 运行服务器的webpack
